# Image Editor

A browser-based image editor built with HTML, CSS, and JavaScript. The application allows users to upload an image, apply real-time filter adjustments, use preset effects, reset edits, and download the edited image directly from the browser.

## Features

* Upload images from your device
* Real-time filter adjustments
* Preset filter effects
* Brightness, Contrast, Saturation, Blur, Hue Rotate, Grayscale, Sepia, Opacity, and Invert controls
* Reset all edits with a single click
* Download the edited image
* Canvas-based image rendering
* Responsive interface for different screen sizes

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* HTML5 Canvas API

## How to Use

1. Click **Choose Image** and select an image from your device.
2. Adjust the available filters using the sliders.
3. Apply a preset effect if desired.
4. Click **Reset** to restore the original image.
5. Click **Download Image** to save the edited image.
